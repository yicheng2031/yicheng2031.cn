# Yicheng Products

Personal product website for:

- PhotoDateStamp - EXIF & Frame
- Xcode Cache Cleaner

The site is plain static HTML/CSS/JS, includes responsive layouts for desktop and mobile screens, and can be deployed directly with GitHub Pages.

## Local preview

```sh
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## GitHub Pages

1. Create a GitHub repository for this folder.
2. Push `index.html`, `styles.css`, `script.js`, `assets/`, and `.nojekyll`.
3. In repository settings, enable Pages from the default branch root.

For a future custom domain, add a `CNAME` file containing `yicheng2031.cn` after DNS is ready.
